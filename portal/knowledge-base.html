Website Composer Guide

Introduction

The Website Composer is a powerful drag-and-drop tool that allows you to create professional websites without writing a single line of code.

Drag & Drop Interface

To start building, simply drag an element from the "Elements" panel on the left and drop it onto your canvas. You can rearrange elements by dragging them to a new position.

Editing Content

Click on any text element to edit it inline. For images and other rich media, selecting the element will open the "Properties" panel on the right where you can update sources and styles.

User Management

Admittance to the portal is controlled via the Organization module.

Roles

  • Admin: Full access to all modules and settings.
  • Editor: Can edit content but cannot change billing or organization settings.
  • Viewer: Read-only access to dashboards.
